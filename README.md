# Sledgehammer
### Created By William McGonagle

## Features
- Custom Mapping System Support
  - Loaded in from custom zip files
  - Functions are run through Lua Files
  - Objects are stored in object files
  - Assets can be stored in the zip files (models, images, sounds)
- Custom Setups
  - *Helps maximize efficiency*
  - Multiple Windows in Different Sizes and Positions
  - Customs buttons that are linked to a command (similar to alias in bash)
  - Extensions similar to google extensions
- Crossplatform
  - Rendering Done with Vulkan
  - Scripts are Lua
- Windows
  - Animation Window
  - Start Window
  - Map Window
  - Texture Window
  - Material Window
  - Mesh Window
- Multi-editing
  - Multiple people can collaborate on the same map
  - Server/ Client Relationship
  - File data is stored on server
